# OnlyBills Privacy Policy Page

A clean, dark-themed privacy policy page for the OnlyBills iOS app.

## Setup

1. Place your App Store logo image as `app-store-logo.png` in the same directory as `privacy-policy.html`
2. Open `privacy-policy.html` in a web browser to preview
3. Deploy to your web hosting service of choice

## Features

- Dark theme with white text
- Responsive design (mobile-friendly)
- Clean, modern Apple-inspired styling
- TL;DR summary section with key points
- Complete privacy policy content
- Social links (GitHub and LinkedIn) in footer
- Automatically updates the "Last Updated" date

## Customization

To replace the app icon placeholder:
- Replace the line `<img src="app-store-logo.png" alt="OnlyBills App Icon" />` with your actual app store logo filename

## File Structure

```
OnlyBills Site/
├── privacy-policy.html  # Main privacy policy page
├── app-store-logo.png   # Your app store logo (add this)
└── README.md           # This file
```

## Deployment

You can host this page on:
- GitHub Pages
- Netlify
- Vercel
- Any static web hosting service

Simply upload the HTML file and your logo image.
